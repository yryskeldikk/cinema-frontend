CSS Part

1.

Link style is used. One stylesheet for common layout and design(header, footer, navigation, logo) is used for each page. Specific styles are applyed as an embedded style.

Using only one stylesheet for common design helps us to save time and we do not have to apply the same style multiple times.

Using embedded style for specific design is much better because it helps us to avoid use of multiple stylesheets for one HTML file.

2. CSS3 Features

2 CSS3 features other than animation are used: border-radius in navigation menu and box-shadow for the table on the "Tickets" page.
